#Netflix Country Analysis

This Python project analyzes the number of titles added to Netflix in 2020 by country.


## Features:
- Cleaned dataset(null values removed, country field standardized)
- Used Pandas and Matplotlib for analysis and visualization
- Visualized top 5 countries with the most added titles in 2020

## Tools:
- Python
- Pandas
- Matplotlib
